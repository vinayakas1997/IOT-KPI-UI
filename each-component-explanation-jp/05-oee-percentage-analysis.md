# OEE分析

## それは何ですか？
設備総合効率（OEE）カードです。稼働率・パフォーマンス・品質の3つのサブ要素と、それらを組み合わせた1つの総合パーセンテージで構成され、ラインが利用可能時間をどれだけうまく使えたかを示す標準的なサマリーです。

稼働率が問うのは「ラインが稼働予定だった時間のうち、実際に稼働していた時間はどれだけか」です。段取り替え、故障、そして計画・計画外を問わずシフトを食いつぶすあらゆる停止によって低下します。

パフォーマンスが問うのは「ラインが稼働している間、定格速度で動いていたか」です。ボトルネック機械が理想サイクルタイムより遅く動いているときは、停止がゼロであっても低下します — これは純粋な速度ロスであり、ダウンタイムではありません。

品質が問うのは「実際に生産したもののうち、一発合格したものはどれだけか」です。スクラップ、手直し、リジェクトによって低下します — これは不良率カードが報告する比率と同じものを、逆の視点で表現したものです。

## それを持つことのKPI価値は何ですか？
総合パーセンテージは、管理者がシフトの設備総合効率を判断するために注視できる唯一の数値ですが、真の価値は診断にあります。3つのサブ要素それぞれが、まったく異なる改善策を指し示すからです。稼働率が低ければダウンタイムと段取り替えを追いかけるべきです。パフォーマンスが低ければ、ボトルネックは稼働していても遅く動いているということなので、サイクルタイム分析とボトルネック機械カードを確認してください。品質が低ければ、機械の速度ではなく不良を追いかけるべきです。

3つの要素は平均ではなく掛け算であるため、他の2つが強くても1つが弱いだけでOEE全体が引きずり下げられます。このダッシュボードの実際の数値では、稼働率73%、パフォーマンス78%、品質86%を組み合わせるとOEEはわずか49%になります — 各要素はそれぞれ単体では「まずまず」に見えても、積によって損失が複合してしまうのです。教科書的な「world class」基準であるOEE85%を達成するには、3つの要素すべてが同時に強くなければならず、通常はそれぞれがおよそ95%を超える必要があります。

稼働率・パフォーマンス・品質は、ライン総合健全性レーダーを構成する6本のスポークのうち3本でもあり、変更を加えずに直接反映されます。そのため、レーダー上のいずれかのスポークが縮んでいる場合、必ずこのカードの特定の行に原因を遡ることができます。品質を含まない「稼働率 × パフォーマンス」だけの値は、計画達成率カードが報告する数値そのものです — つまり計画達成率は「生産目標を達成できたか」に答え、OEEはさらに「作ったものは実際に良品だったか」にも答えます。

## それを計算するために何が必要ですか？
稼働率 = 稼働時間 / 計画生産時間。したがって計画スケジュールに対するダウンタイムのみが影響します。

パフォーマンス = ボトルネックの理想サイクルタイム / 実際のサイクルタイム。ボトルネックのペースがライン全体の生産量の上限を決めるため、特にボトルネックについて測定します — これは設備稼働率カードで説明されているブロック・待機による待ち時間で押し上げられる、同じ実サイクルタイムです。

品質 = 承認済み総生産数 / (承認済み総生産数 + 不良数) — 不良率カードが示す比率の逆数です。

OEE % = 稼働率 × パフォーマンス × 品質。これは3つのパーセンテージの平均ではなく、3つの分数の積です — そのため1つの弱い要素が最終数値にこれほど大きなコストをもたらすのです。

# 時間別エラー発生パターン（件数と時間内タイミング）

## それは何ですか？
区間ごとのエラー件数と、その区間内で各機械のエラーが正確にいつ発生したかを示すタイミングトラックを組み合わせたチャートで、機械／人為／その他に分類されます。

## それを持つことのKPI価値は何ですか？
区間内タイミングは、単純な件数集計では見えないクラスタリングのパターンを明らかにし、カテゴリ分類によって設備の問題と教育・プロセスの問題を切り分けます。

## それを計算するために何が必要ですか？
共有のダウンタイム／アラームイベントリストから導出されます。各イベントの開始時刻はシフト区間に振り分けられ、その区間の実際の長さに対するオフセットとして位置づけられます。

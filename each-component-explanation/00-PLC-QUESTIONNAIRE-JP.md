# PLC能力アンケート — 条件付き (PLC Capability Questionnaire — Conditional)

**目的 (Purpose)**: PLCチームが記入して返送してください。質問は条件付きです — 直接レジスタ (Method A) がある場合は、ビットレベルの質問 (Method B) をスキップしてください。

---

## ライン構成 (Line Architecture)

```
材料 (Material) → [機械1] → [機械2] → [機械3] → [機械4] → [機械5] → [品質ゲート (Quality Gate)] → 出力 (Output)
```

**クイックリファレンス (Quick reference)**:
- **Method A** = 直接レジスタ (Direct register) → ダッシュボードが直接読み取ります。常に優先。
- **Method B** = 生ビット/信号 (Raw bits/signals) → ダッシュボードが値を導出します。Method Aが利用できない場合の代替手段。

## PLCチーム向け手順 (Instructions for PLC Team)

1. 各質問に **Y** (既に利用可能) または **N** (利用不可) で回答してください
2. 各 **Y** について、ダッシュボードが読み取るべきレジスタ名またはビット名を記入してください
3. セクション内のMethod Aの質問に **Y** と回答した場合、そのセクションのMethod Bの質問はスキップしてください
4. このフォームをダッシュボードチームに返送してください
5. 利用可能なレジスタをダッシュボード機能にマッピングし、結果を報告します
